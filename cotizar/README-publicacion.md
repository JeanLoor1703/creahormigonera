# Publicacion de la pagina de cotizacion

Esta carpeta ya esta lista para subirse como sitio estatico.

## Archivos principales

- `index.html`
- `styles.css`
- `script.js`
- `assets/creacom-hormigon.svg`

## Opcion 1: Subirla completa a Netlify o Vercel

1. Comprime el contenido de esta carpeta o subelo a un repositorio.
2. En Netlify o Vercel, crea un nuevo sitio.
3. Sube esta carpeta completa o conecta el repositorio.
4. El archivo de entrada es `index.html`.

No necesita backend ni base de datos para funcionar.

## Opcion 2: Agregarla a la web actual

Si ya tienen una web, pueden:

1. Crear una carpeta como `/cotiza/` o `/cotizacion/` dentro del sitio.
2. Copiar estos archivos dentro de esa carpeta.
3. Abrir la pagina desde una ruta como:

`https://tudominio.com/cotiza/`

## Importante

- El logo ya esta dentro del proyecto y no depende de archivos locales.
- El boton de WhatsApp ya usa el numero `0997 528 670`.
- Si cambian de dominio o estructura, no hace falta tocar nada mientras se mantengan juntos estos archivos.
